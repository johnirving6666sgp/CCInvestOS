## P1 · 港股18A裸泳榜单 — 死亡模式四特征（华创会，2026-06-12）

**来源**：用户分享文章，2026-06-13
**核心警句**："烧钱换未来的公司，最怕未来不来。"

### 四条死亡模式：
1. **烧钱多年，商业化遥遥无期** — 上市2-5年，全部亏损，无一实现年度盈利
2. **核心产品竞争力被稀释** — 进度不及预期，或竞品已获批上市，技术不再"唯一"
3. **流动性枯竭** — 日成交额极低，卖盘稍大即瀑布式下跌，丧失定价权
4. **再融资困局** — 股权融资不可能，银行不看，只能等死或被并购

### 典型案例：
- 永泰生物：市值3.58亿，跌幅80%，现金<2年，6年无商业化产品
- 圣诺医药：市值4.90亿，竞品已获批，现金撑不到2027年
- 药明巨诺：CAR-T销售额不足2亿元，远不及预期，从200亿跌至零头
- 腾盛博药：新冠产品归零，其他管线早期

## P2 · Serenity"瓶颈理论"（Chokepoint Theory）— 稀缺性投研方法论（华创研究院，2026-06-12）

**来源**：用户分享文章（华创会，叶灿斌/潘思琪），2026-06-13

### 核心思想：
**谁最难替代、最难被绕过，谁就截留供应链中最丰厚的利润。**
刻意回避热门龙头，聚焦市值100-500亿、低换手率、被错误定价的供应链卡点公司。

### 四种稀缺性机制：

**① 产能约束**
- 典型：HBM，从启动扩产到产能落地32个月以上，长协锁单排到2028年后
- 估值重构：周期股（6-8x PE）→ 成长股（15-20x PE），驱动因素是现金流可见度提升+排他性

**② 原材料垄断**（最接近物理限制）
- 典型：InP衬底，日本住友电工占全球60%+，供需缺口超70%
- Lumentum是全球唯一可规模量产200G/lane EML的公司，订单排到2026年底
- 国内：云南锗业（002428），华为锁定约53%产能，但市场仍用周期股框架定价
- **注意力永远先在组装层，后在芯片层，再往上到材料层——这个迟滞本身是机会**

**③ 代际差（技术过渡窗口）**
- 典型：CPO推迟至2028-2029 → NPO/LPO/pluggable过渡需求反而更强
- 市场错误：把"CPO是终局"混同于"过渡窗口不值钱"→ 跌停板是错杀
- 估值逻辑：过渡期标的不应按周期股PE定价，应按渗透率曲线×单价×数量建模

**④ 国产化率不足**
- 典型：长鑫存储，全球DRAM份额<10%但季度份额跳变（4.7%→7.67%，一个季度+3pct）
- 内部垄断结构：国内DRAM IDM量产只有长鑫一家，HBM国产也只有长鑫
- 传导机制：国际叙事形成后，国内对标标的估值弹性更容易打出来

### 闭环操作方法：
1. 先确认大趋势（AI硬科技 vs 消费赛道分水岭）
2. 从终端用户需求反推上游，画供应链图谱，找价值洼地和卡点
3. 多维交叉验证：订单数据、客户结构、认证进度、扩产节奏、财报措辞
4. 持续追踪动态：区分什么在证实判断、什么在证伪判断
5. 仓位与认知匹配：举证充分、逻辑自洽才重仓

### 三个失效条件（反脆弱性）：
1. AI资本开支2026-2027年明显收缩（最重要指标：谷歌/微软/亚马逊/Meta季度capex）
2. CPO量产时点提前至2026年下半年（过渡窗口提前关闭）
3. InP国产替代加速，云南锗业/三安光电良率2027年达到住友电工标准
