# 投资模式库

> 把你脑中可复用的投资逻辑代码化。每个模式 = 一句话本质 + 筛选条件 + 证伪条件。状态 [启用] 的模式会被猎手引擎每周用于全市场狩猎。对话中说「存为模式:……」即可新增,说「修改模式 P1 ……」即可调整。

## P1 · 王者归来 [用户原创 · 启用]

- **一句话**:曾经的王者跌落深谷,基本面出现反转信号,而股价仍在低位——市场还没相信它回来了。
- **筛选条件**:
  1. (硬前提)股价距历史/52周高点回撤 ≥40%,处于 52 周区间下三分之一
  2. 曾是公认的行业王者:品牌、渠道、技术资产仍在
  3. 反转信号至少一项有实据:新管理层 / 新产品周期 / 行业出清 / 盈利拐点
  4. 市场仍在怀疑:卖方情绪冷淡或评级偏空(逆向前提)
- **证伪条件**:反转信号连续两个季度落空;资产负债表恶化危及生存
- **历史案例**:Palantir（美股）、汇聚科技01729（港股）、诺诚健华（港股18A候选）

## P2 · 戴维斯双击 [经典 · 待确认]

- **一句话**:盈利增长与估值修复同时发生,收益被双重放大。
- **筛选条件**:盈利增速进入上行通道;当前估值低于历史中枢与同业;存在催化剂让市场重新定价
- **证伪条件**:盈利增速回落;估值修复完成后透支

## P3 · 错杀成长 [经典 · 待确认]

- **一句话**:长逻辑未变的成长股,因短期事件被市场过度抛售。
- **筛选条件**:回撤由一次性/短期事件触发;核心增长引擎(用户数/订单/份额)未受损;现金流或融资足以穿越
- **证伪条件**:"一次性事件"被证明是结构性问题

## P4 · Chokepoint供应链卡点 [Serenity方法论 · 启用]

- **一句话**:不买最亮的星，买星光必须经过的那道门——供应链中无法绕开的物理或技术卡点，处于机构盲区，被错误定价。
- **筛选条件**:
  1. (硬前提)市值100-500亿美元（机构盲区区间，过大则已被发现）
  2. 处于供应链的物理/技术卡点：客户无法绕过、替代成本极高
  3. 低换手率、分析师覆盖稀少（机构盲区的外部信号）
  4. 股价处于52周区间下三分之一或合理中枢（非高位追入）
  5. 行业整体资本开支处于扩张期（需求侧有顺风）
- **证伪条件**:卡点被技术路线替代（如光纤被铜缆重新取代）；客户自研垂直整合绕过该环节；市值超过500亿后机构已充分覆盖
- **与P1组合**:Chokepoint公司若同时满足P1（股价低位+反转信号），是最高优先级介入机会
- **当前观察池（AI供应链）**:
  - CLS（Celestica）：机架集成ODM，市值$100亿，77%高位，等回调
  - COHR（Coherent Corp）：光学收发器，市值$200亿，88%高位，等回调
  - PWR（Quanta Services）：电力互联施工，市值$440亿，82%高位，框架上限
  - IRM（Iron Mountain）：标签错误定价的数据中心转型，市值$370亿，91%高位
  - NBIS（Nebius Group）：欧洲AI云基础设施，需重新评估市值与位置
  - CRDO（Credo Technology）：有源电缆/SerDes，市值$220亿已超框架，已被机构发现
  - ALAB（Astera Labs）：PCIe重定时器，52周高位100%，已被机构发现
