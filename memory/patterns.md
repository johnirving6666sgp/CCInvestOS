# 投资模式库

> 把你脑中可复用的投资逻辑代码化。每个模式 = 一句话本质 + 筛选条件 + 证伪条件。状态 [启用] 的模式会被猎手引擎每周用于全市场狩猎。对话中说「存为模式:……」即可新增,说「修改模式 P1 ……」即可调整。

## P1 · 王者归来 [用户原创 · 启用]

- **一句话**:曾经的王者跌落深谷,基本面出现反转信号,而股价仍在低位——市场还没相信它回来了。
- **筛选条件**:
  1. (硬前提)股价距历史/52周高点回撤 ≥40%,处于 52 周区间下三分之一
  2. 曾是公认的行业王者:品牌、渠道、技术资产仍在
  3. 反转信号至少一项有实据:新管理层 / 新产品周期 / 行业出清 / 盈利拐点
  4. 市场仍在怀疑:卖方情绪冷淡或评级偏空(逆向前提)
- **证伪条件**:反转信号连续两个季度落空;资产负债表恶化危及生存
- **历史案例**:Palantir（美股）、汇聚科技01729（港股）

## P2 · 戴维斯双击 [经典 · 待确认]

- **一句话**:盈利增长与估值修复同时发生,收益被双重放大。
- **筛选条件**:盈利增速进入上行通道;当前估值低于历史中枢与同业;存在催化剂让市场重新定价
- **证伪条件**:盈利增速回落;估值修复完成后透支

---

## P3 · 错杀成长 [经典 · 待确认]

- **一句话**:长逻辑未变的成长股,因短期事件被市场过度抛售。
- **筛选条件**:回撤由一次性/短期事件触发;核心增长引擎(用户数/订单/份额)未受损;现金流或融资足以穿越
- **证伪条件**:"一次性事件"被证明是结构性问题

---

## P4 · 瓶颈猎手（Chokepoint Hunter）[Serenity方法·用户分享·启用]

- **一句话**:在机构盲区中找到产业链的真实物理卡点——不热、不大、但不可绕过。
- **筛选条件**:
  1. 市值100-500亿区间（机构覆盖稀疏的盲区）
  2. 低换手率、卖方评级稀少或冷淡（逆向前提：尚未被发现）
  3. 处于供应链某个关键物理环节，客户难以短期切换（真实卡点，非概念）
  4. 股价处于52周区间下半段（低位介入，赔率合理）
  5. 行业资本开支周期明确向上，需求端有外部驱动（顺风）
- **证伪条件**:卡点被技术替代（如光电共封装彻底替代外置光模块）；大客户自研垂直整合；市值突破500亿后机构全面覆盖，盲区消失
- **历史案例**:汇聚科技01729（AI数据中心光纤线缆卡点，2022年介入）
- **当前候选观察池**:CLS、COHR、PWR、IRM、NBIS（均处于高位，等待回调）；CRDO/ALAB逻辑强但已超出盲区
- **来源**:用户2026-06-13分享华创研究院文章，Serenity瓶颈理论
